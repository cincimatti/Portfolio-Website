Software Projects

The software collection below represents several projects developed while enrolled in the MIT Full Stack Development with MERN course. The MERN stack represents a set of frameworks used in web developement including Mongo, Express, React, & Node technologies, while also leveraging HTML, CSS, and JavaScript.

PacMen Exercise

This webapp mimics game functions. It demonstrates the use of Arrays, Functions, and Object movement within the DOM using JavaScript.

Eye Exercise

This webapp uses HTML/CSS with JavaScript to move on-screen elements with mouse movements. The result is eyes that watch your every move!

Bus Tracker

This is a geospatial webapp that uses JavaScript to plot bus stops along geocoordinates. The application uses Mapbox as the backend API for mapping.
